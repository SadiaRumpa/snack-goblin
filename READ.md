# 🧌 Snack Goblin

A cute, chaotic mobile arcade game where you play as a goblin trying to eat junk food while dodging healthy vegetables!

## How to Play

1. Open `index.html` in any modern web browser (works great on mobile!).
2. Tap "START MUNCHING".
3. Drag your finger (or mouse) left and right to move the goblin.
4. Collect 🍪 🥤 🍟 🍰 for +10 points.
5. Avoid 🥦 🌶️ 🥕 or lose a life!
6. The game gets faster every 100 points.

## Tech Stack

- HTML5 Canvas
- CSS3
- Vanilla JavaScript

## Deployment

This game is ready to be hosted on GitHub Pages. Just push to a repository and enable Pages in the settings!
